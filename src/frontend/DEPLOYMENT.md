# ByteWay Deployment Guide

This guide explains how to publish ByteWay to the Internet Computer (IC) mainnet.

## Prerequisites

- [DFX SDK](https://internetcomputer.org/docs/current/developer-docs/setup/install/) installed (version 0.15.0 or later)
- Cycles wallet configured for mainnet deployment
- Admin Internet Identity created
